## What is this repository for?

- This repo **does not contain blog source code**
- It is only used to store **blog comments** via:
  - GitHub Discussions
  - giscus (comment system for static sites)

Each blog post maps to a discussion thread automatically.

---

## How comments work

- Comments are stored as **GitHub Discussions**
- Authentication is handled via **GitHub login**
- The comment widget is embedded on the blog using **giscus**
- No third-party comment platform (e.g. Disqus) is involved
